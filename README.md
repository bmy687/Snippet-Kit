Code Performer - VS Code 演示神器
提前保存代码片段，现场演示时随便按键盘，输出的就是你预设的代码。

功能
    保存代码片段：提前把要演示的代码粘贴进去，取个名字保存
    一键演示：选择一个片段开始演示，随便按键盘任意键，输出的都是预设代码
    隐藏面板：演示时自动隐藏管理面板，按 Ctrl+Alt+H 可随时切换显示/隐藏
    编辑/删除：随时管理已保存的片段
    自然模拟：每次按键随机输出1-3个字符，模拟真实打字的节奏
    安装方法
    方法一：直接安装 .vsix 文件
    在 VS Code 中按 Ctrl+Shift+P
    输入 Extensions: Install from VSIX...
    选择 code-performer-1.0.0.vsix 文件
    安装完成！


使用方法
    第一步：保存代码片段（提前准备）
    点击左侧活动栏的 Code Performer 图标（播放按钮）
    在「添加代码片段」区域：
    输入片段名称（如"登录模块"）
    选择编程语言
    粘贴你要演示的代码
    点击「保存片段」
    可以保存多个片段，按演示顺序依次使用。

第二步：现场演示
    打开一个空白文件
    在侧边栏找到要演示的片段，点击「开始演示」
    或者按 Ctrl+Shift+P → 输入 Code Performer: 选择片段开始演示
    侧边栏会自动隐藏
    现在随便按键盘上任意键，屏幕上就会输出你预设的代码
    底部状态栏会显示演示进度
    第三步：结束演示
    代码输出完毕后会自动结束
    也可以点击底部状态栏的演示提示来手动停止
    或按 Ctrl+Shift+P → Code Performer: 停止演示
    快捷键
    快捷键	功能
    Ctrl+Alt+H	切换面板显示/隐藏
    命令
    命令	说明
    Code Performer: 打开管理面板	显示侧边栏面板
    Code Performer: 隐藏管理面板	隐藏侧边栏面板
    Code Performer: 选择片段开始演示	从列表中选择片段开始
    Code Performer: 停止演示	停止当前演示

注意事项
    演示时你按什么键都无所谓，每按一次就会输出预设代码的下一个字符
    演示时侧边栏会自动隐藏，领导看不到你的片段列表
    建议演示前先新建一个空文件
    多个片段可以按顺序依次演示，演完一个再选下一个
